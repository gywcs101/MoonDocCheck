# moon_source_scan.mbt

## 文件职责

扫描 `.mbt` 源码文本，识别公开 API 声明，匹配声明前的 `///` 文档注释，并标记缺失文档或弱文档。

## 当前完成情况

- [x] 支持常见公开声明识别。
- [x] 支持 `pub(all)` 和 `declare pub`。
- [x] 支持声明名称提取。
- [x] 支持连续 `///` 文档注释匹配。
- [x] 支持弱文档初步检测。

## 待完善

- [ ] 支持多行声明。
- [ ] 支持声明前属性。
- [ ] 与 `.mbti` 结果对照。

## 当前问题

- 当前是文本扫描，不是完整 MoonBit 解析器；复杂语法可能需要后续增强。
